# 문자 변수

 - 문자(열)을 표현하고자 할 때는 문자를 "" 또는 '' 안에 입력해야 함.
 
 - 내장 함수를 사용하여 문자 개수, 문자열분리, 문자열 합치기, 대소문자 변환 등을 할 수       있음.
 - 문자 개수 확인 
    nchar(변수명)
 - 문자열 합치기 
    paste(문자열1,문자열2,sep="문자열 사이 기호")
    paste(문자열,collapse="문자열 사이 기호")
    
    sep 를 생략하면 항목 사이에 공백을 줌(벡터 간 결합에 사용)
    collapse : 문자열 내 구분자 넣기 
    
# 범주형 변수 : 팩터(Factor)
 - 범주형 데이터를 표현함, 값이 대상을 분류하는 의미를 지니는 변수 
 - 제한된 범주 안에서 선택된 데이터 
   ex) 성별,수강과목,혈액형 등 
 - 범주 별 비교분석, 범주 이외의 값 설정 통제 가능 
 - 서열형 데이터 
   만족도 설문 조사 응답 자료, 대학 성적과 같이 항목간에 서열이 존재함 
 - 명목형 데이터
   단순분류
 - 숫자가 크기를 의미하지 않으므로 더하거나 평균을 구하는 산술을 할 수 없음.
